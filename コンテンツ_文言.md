### アプリ1: アトデヤル
```
名前: アトデヤル
タグライン: 先延ばしを減らす、シンプルなToDoアプリ
説明: 「今やること」を1つだけ前面に出すことで、タスクの多さに迷わず行動を始めやすくするiOSアプリ。タスク一覧を眺めて何もできない、という状況を減らしたくて作り始めた。
課題・目的: ToDoリストにタスクが並ぶほど、何から手をつけるか迷って先延ばしになってしまう。「今これをやる」の1点に絞る仕組みがあれば動きやすくなると思った。
工夫した点: シンプルに見せながら、必要な情報を適切なタイミングで出す設計を意識している。起動してすぐ動けるUXを目指している。
使用技術: Swift / SwiftUI
ステータス: 開発中
アイコン: /apps/atodevaru.png
リンク: なし（「開発中につきリンク準備中」と表示）
```

### アプリ2: 生き急げ！
```
名前: 生き急げ！
タグライン: 時間の有限さを意識して、動くためのアプリ
説明: 人生や1日の残り時間を可視化することで、「まあいいか」ではなく「今やろう」と思えるようにするiOSアプリ。時間への意識を高める仕掛けをシンプルなUIで実現したい。
課題・目的: 時間は有限とわかっていても、なかなか実感しにくい。数字や視覚的な表現で「残り時間」をリアルに感じられる仕組みがあれば行動が変わると思った。
工夫した点: 焦らせすぎず、でも時間の感覚を適切に刺激するバランスのUI設計が難しい。使うたびに少しだけ意識が変わる体験を目指している。
使用技術: Swift / SwiftUI
ステータス: 開発中
アイコン: /apps/ikiisouge.png
リンク: なし（「開発中につきリンク準備中」と表示）
```
